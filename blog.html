DOM stands for Document Object Model. It is a method to make the web page interactive. We can manipulate any HTML element as well can style them with the help of DOM.

We can select the HTML using for DOM manipulation using different methods. Such as

  • document.getElementsByTagName()

  • document.getElementsByClassName()

  • document.getElementById()

  • document.querySelector()

  • document.querySelectorAll()

The Event bubbling is a kind of event propagation. This event triggers the child elements of the parent elements in a HTML file. Then from the targeted element it consecutively triggers the parent element of the target element then go upwards until it reach the highest parent.

A callback function is a function which is called by another function. In other words, it is accessible in another function. Sometimes we need to some repetitive task in some other functions then in that case we will make a common function for those same task, and will call that common function inside function the where we have our repetitive tasks.
